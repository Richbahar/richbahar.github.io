首页 文章

文学作品的鉴赏得置于时代背景之中，然后与作者生平勾联。然而此次，我对历史背景将不做阐述，作者生平与时代背景百科可考，为了节省篇幅，也不做过多介绍。

这篇笔记是以宋代文学的创作者为主线，介绍其创作风格及对文学发展的贡献，偶尔会摘录代表性作品。

# 北宋初期文学

北宋初期文学处于沿袭躺五代文风的过渡阶段，为宋代文学的发展作了铺垫：

  • 柳开等人倡导和写作的古文；
  • 王禹偁革新诗风和文风；
  • 杨亿等人的西昆体；
  • 晏殊令词创作
# 柳开

著《河东先生集》

反对五代浮艳文风，回复 “韩柳” 古文传统。主张文道合一，提倡复古、简朴文风。

# 穆修

柳开之后的古文作家。

强调教化作用，力求结构章法变化，多用散行单句，不堆砌文藻，不追球艳丽。

# 王禹偁

诗歌平淡清远，为文古雅简淡。代表作《小畜集》

他散文表现出的平易自然的风格成为欧阳修诗文改革运动的先绳。

作品分期：

  • 早年：学习白居易的闲适唱和诗；
  • 贬官后：学习白居易讽谕诗，更进学杜甫诗风

七言古诗学习白居易的平易风格；

近体诗、绝句尚有清淡清远的格调

倡导古文写作，反对五代的雕绘风气，创作古雅简淡的作品。其中多有现实的政治内容和鲜明的思想倾向。

# 杨亿

西昆体代表诗人，与王钦若等编撰《册府元龟》。

作诗讲究辞采，风格典丽，表现才学和功力，也有清峭感怆的讽谕之作。

# 林逋

“梅妻鹤子”，以江湖散人子诗装点山林，用细碎小巧的笔法写清苦幽静的隐居生活。

诗歌风格：孤淡清逸。

《山院小梅》

众芳摇落独暄妍，占尽风情向小园。 疏影横斜水清浅，暗香浮动月黄昏。

霜禽欲下先偷眼，粉蝶如知合断魂。 幸有微吟可相狎，不须檀板共金樽。

# 晏殊

继承唐五代词的艺术经验，作词在语言上明净雅致，构思上曲折精巧，内容上表现了情感经历和对人生的思考，直接影响欧阳修、晏几道等人。

风格：含蓄委婉

代表作《珠玉词》《晏元献遗文》（清代人所著）

《踏莎行》

小径红稀，芳郊绿遍，高台树色阴阴见。春风不解禁杨花，蒙蒙乱扑行人面。

翠叶藏莺，朱帘隔燕，炉香静逐游丝转。一场愁梦酒醒时，斜阳却照深深院。

# 北宋诗文革新

北宋中叶的古文运动出了确定古文在散文领域的正宗地位，而且作诗也讲究 “气格”、“以文为诗”。其中欧阳修是这场运动的文坛盟主，他对众文人的推崇，有力推动了北宋文学的发展，使宋诗、词自具面目，宋代士风和文风从根本上开始转变。

# 欧阳修

北宋诗文革新运动的领导者，继承发扬韩愈的古文理论，开创一代文风，同时也对诗风和词风进行了革新。

文学贡献：

  • 文

    散文：学习韩愈，发展 “文从字顺”，言简而明、信而通。

    特点：从具体事物出发，写得婉转曲折、含蓄蕴藉。同时又使得文章富有情感和 诗味。

    代表作《醉翁亭记》

    赋：开启宋赋先河，变骈偶对仗为骈散相间、奇偶错杂。保留赋铺陈叙述的特性，又让文章趋于和缓。

  • 诗

    • 学习韩愈的 “以文为诗”，诗歌创作散文化
    • 追求气格
    • 开创了取材广泛、命题新颖、以文为诗及议论为诗的风格

  • 词

    与晏殊同为北宋初花间词人代表，除占比例较大的情词外，也有抒发旷达胸怀，风格豪放的作品。

    《浪淘沙》

    把酒祝东风，且共从容。垂杨紫陌洛城东。总是当时携手处，游遍芳丛。
    聚散苦匆匆，此恨无穷。今年花胜去年红。可惜明年花更好，知与谁同？

# 梅尧臣

与苏舜钦齐名 “苏梅”，著《宛陵先生文集》。

艺术特点：平淡其表，深邃其里

  • 诗歌 “古淡”，社会矛盾与个人身事激变时，趋于劲峭苦硬；
  • 宦途不顺归隐山林，则表现为冲和恬淡，蕴含深意

《鲁山山行》

适与野情惬，千山高复低。好峰随处改，幽径独行迷。

霜落熊升树，林空鹿饮溪。人家在何许？云外一声鸡。

# 苏舜钦

采用豪犷激切、直抒胸臆的方式写诗：

  • 前期：揭露和抨击时弊，有奋不顾身的气概，也有粗燥、生硬，缺乏隽永的缺点。
  • 后期：诗风多变，寄情山水，有幽独闲放的趣味。

《淮中晚泊犊头》

春阴垂野草青青，时有幽花一树明。

晚泊孤舟古祠下，满川风雨看潮生。

# 王安石

创造 “半山体”（又名” 王荆公体 “），有《王临川集》《临川集拾遗》

文学成就：

  • 文：重以致用，简练明快，笔力雄健
    • 政论文：严谨，语言朴素简洁，说理透彻；
    • 散文：议论说理见长；
    • 小品文：笔力雄健，富有感情，文风峭刻；
    • 议论文：结构严谨，析理透辟；
    • 叙事抒情文：随笔挥洒，曲尽其妙。
  • 诗词：化奇崛于寻常，既清新闲适，又沉郁悲壮，被称为 “半山体”
    • 早期：直抒胸臆，意味较薄
    • 晚年：以绝句为主，新颖别致，雅丽精绝，形成 “深婉不迫‘的风格，体验宋诗风貌，又有唐诗复归倾向。

《书湖阴先生壁》

茅檐长扫净无苔，花木成畦手自栽。
一水护田将绿绕，两山排闼送青来。

# 曾巩

世称 “南丰先生”，文章 “古雅、平和、正直 “，著《元丰类稿》。

文学特点：

  • 追求雍容典雅的艺术风格，文章冲和平淡，委屈周详；
  • 长于议论，引经据典，明白详尽；
  • 布局谨严，节奏舒缓，语言干净，思致明晰。

代表作 《墨池记》

# 苏洵

苏洵文章以学养为基础，重气势，有战国纵横家色彩，擅长散文，尤其擅长政论文。著《嘉祐集》。

写作特点：结构谨严、说理周详、气势磅礴、曲折多变。

文风：精练劲建、简古质朴；笔锋老辣，纵横博辩。

# 苏辙

学习欧阳修淡泊平和文风，议论以稳健见长。著有《栾城集》。

作品委婉纡徐的艺术情趣融入其中，苏轼称其散文 “汪洋淡泊，有一唱三叹之声，而其秀杰之气终不可没”。

# 苏轼

文学成就：

  • 散文成就

    • 议论文
      • 纵横捭阖，气势充沛，雄辩恣肆；
      • 议论透辟，通俗明晓，深入浅出，文采斐然。
    • 记叙文
      • 文学价值最高，最具独创性的，以山水游记和亭台堂阁记为代表；
      • 游记表现出借景立论的特点；
      • 记叙文为文学性散文，展现苏轼随物赋形的本领，体现他才情横溢的奇思妙想

  • 诗歌创作

    • 创作概况
      • 主要分为：社会政治诗、山水景物诗、和陶诗、题画诗
      • 以贬谪黄州为界限，早期写富有批判精神的社会政事诗，后期则抒发被贬后的感慨。
    • 创作特色
      • “以文为诗”，以议论为诗，将事物形象感受与哲理思考结合；
      • 创造 “东坡体”，擅长七言，风格多样；
      • 基本风格：
        • 刚健含婀娜的清丽雄健；
        • 豪放加平淡的清旷闲逸。

  • 词的革新意义

    • 超旷高远胸襟

      “以诗为词”，用诗的品格改造传统本色的词体，用诗的精神提高词的品味

    • 题材的拓展

      • 士大夫文人的生活内容带到词中；
      • 将情感范围扩大到朋友、师生、兄弟、夫妻之间；
      • 农村题材正式入词
      • 咏物词有开拓之功

    • 豪放旷达的风格

      • 打破了以婉丽柔媚为美的局限，抒发士大夫情致的创作心理；
      • 苏词气象宏大，雄健清刚，创新词风，突破音律束缚。

《冬景》

荷尽已无擎雨盖，菊残犹有傲霜枝。

一年好景君须记，最是橙黄橘绿时。

《望江南》

春未老，风细柳斜斜。试上超然台上看，半壕春水一城花，烟雨暗千家。

寒食后，酒醒却咨嗟。休对故人思故国，且将新火试新茶，诗酒趁年月。

# 江西诗派

江西诗派是以黄庭坚为宗主、对宋代影响最大的诗歌流派，他们对宋诗的发展有很大贡献。

# 黄庭坚

著《山谷集》《山谷外集》《山谷别集》。

题材内容：

  • 日常交际应酬的赠答诗、次韵诗、咏物诗；
  • 反映民生疾苦和时事政治的诗，直接涉及现实；
  • 表现自我人格和襟怀的抒情写意诗。

诗歌特色：

  • 学习杜甫诗歌，多用拗句；
  • 以学问为诗，以故为新，变俗为雅，追求生新瘦硬，工于练字（善用动词、形容词、语助虚词）
  • 创造 “山谷体”
    • 基本特征：求生避熟，求雅脱俗
    • 艺术特点：
      • 学杜为宗旨，重视句法，发展拗句、拗律的体制；
      • 强调 “无一字无来处”，用典以故为新、变俗为雅，长于点化铸造，富有思致和机趣；
      • 好奇尚硬，造语洗净铅华，独树隽旨，风格生新瘦硬峭拔，兼备浏亮芊绵的一面。

词：

雅俗并存：

  • 一方面词名颇高，但词风俚俗；
  • 雅词带有兀拗俊洁的个性色彩。

《寄黄几复》

我居北海君南海，寄雁传书谢不能。桃李春风一杯酒，江湖夜雨十年灯。
持家但有四立壁，治病不蕲三折肱。想见读书头已白，隔溪猿哭瘴溪藤。

《清平乐》

春归何处。寂寞无行路。若有人知春去处。唤取归来同住。

春无踪迹谁知。除非问取黄鹂。百啭无人能解，因风飞过蔷薇。

# 陈师道

作诗凭学力专精，讲苦吟，求奇拙。

创作力求精简字句，摒弃华辞丽藻，其诗无华无才，只有瘦硬。

创作 “后山体” 诗风。

# 陈与义

创造 “简斋体”，著有《简斋集》。

诗歌创作：

  • 前期：内容多为抒怀、咏物、唱和的作品，风格清新冲淡，明净雅致。
  • 后期：内容多为忧国感时，诗风雄阔浑厚，沉郁悲壮。

简斋体：

  • 指他后期创作的诗歌；
  • 对仗不注重字面的工稳精巧，注重上下句的气脉联系，让诗的意境更加圆融；
  • 诗风遒劲，思力沉挚，取材和诗境恢弘，形成了雄浑、沉郁的独特艺术风格。

# 北宋词坛

可分为三个时期：

  • 北宋词初期：继承花间派与南唐词风而有变化
  • 北宋词创造时期：出现了一批推动宋词发展的关键人物
  • 北宋词总结期：集大成者的周邦彦与南渡词人
# 柳永

北宋最有名 “通俗歌曲作家”，北宋和南宋诗词承先启后的词人，著《乐章集》。

题材内容：

  • 表现下层市民情感、观念、价值标准；
  • 言情词，描写风尘女子与艳情；
  • 羁旅行役词；
  • 城市风光；

柳永词的开拓意义：

  • 继承了民间词的传统；
  • 扩大了题材内容（主要在于文化下移）；

对宋词的贡献：

  • 创制慢词，完善慢词体制和表现手法；
  • 章法上，注重叙事完整，描写细腻的效果，语言通俗（柳词广泛流传的根本原因）；
  • 韵脚平仄的变化，吸引和感染观者；
  • “曲祖” 称号

《八声甘州》

对潇潇暮雨洒江天，一番洗清秋。渐霜风凄紧，关河冷落，残照当楼。是处红衰翠减，苒苒物华休。唯有长江水，无语东流。
不忍登高临远，望故乡渺邈，归思难收。叹年来踪迹，何事苦淹留？想佳人妆楼颙望，误几回、天际识归舟。争知我，倚阑杆处，正恁凝愁！

# 秦观

与柳永合称 “秦柳”，与周邦彦合称 “秦周”，著《淮海词》（又名淮海居士长短句）

题材：相思离别的情感；

音律上：平仄协调、音节和谐、节奏鲜明，具有音乐美；

特点：

  • 清新雅致又明白晓畅
  • 抒情中融入议论，委婉深沉又自然流畅

《减字木兰花》

天涯旧恨，独自凄凉人不问。欲见回肠，断尽金炉小篆香。
黛蛾长敛，任是春风吹不展。困倚危楼，过尽飞鸿字字愁。

# 晏几道

著有《小山词》。

题材：爱情生活

词风：语言清丽、感情真挚

《临江仙》

梦后楼台高锁，酒醒帘幕低垂。去年春恨却来时。落花人独立，微雨燕双飞。
记得小蘋初见，两重心字罗衣。琵琶弦上说相思。当时明月在，曾照彩云归。

# 贺铸

俗称 “贺龟头”，诗集《庆湖遗老集》，词集《东山词》。

艺术特点：

  • 盛丽、妖冶者继承花间词风，如《青玉案》，得到 “贺梅子” 称号；
  • 抑塞不平之气时于笔端发之。
# 周邦彦

“正宗”、“集大成” 者，与柳永并称 “周柳”，著有《清真集》《片玉集》。

题材内容：男女之情，相思离别。

艺术成就：

  • 继承柳永铺叙的特点，曲折变化；
  • 章法布局开阖回旋，既有柳永的叙事容量和感情力度，又避免了直露无余，章法布局上常使用不同感情的相互衬托手法；以景结尾。
  • 语言典雅、音律精严，融前人诗词入句。

《旧游・记愁横浅黛》

记愁横浅黛，泪洗红铅，门掩秋宵。坠叶惊离思，听寒螿夜泣，乱雨潇潇。凤钗半脱云鬓，窗影烛光摇。渐暗竹敲凉，疏萤照晚，两地魂消。

迢迢，问音信，道径底花阴，时认鸣镳。也拟临朱户，叹因郎憔悴，羞见郎招。旧巢更有新燕，杨柳拂河桥。但满目京尘，东风竟日吹露桃。

# 李清照

著有《漱玉词》。

《词论》

  • 一是尚雅：指思想的雅正和语言音律的文雅；
  • 二是主张 “词别是一家”，诗词有别。

创作风格：

  • 前期：绰约轻倩，自然妩媚，清丽淡雅；
  • 后期：沉郁哀痛。

创作特点：

  • “易安体”；
  • 语言清新淡雅又通俗易晓；
  • 善于将语言变化与声调、词情相结合。

《武陵春・春晚》

风住尘香花已尽，日晚倦梳头。物是人非事事休，欲语泪先流。
闻说双溪春尚好，也拟泛轻舟，只恐双溪舴艋舟，载不动许多愁。

# 朱敦儒

有词三卷，著为《樵歌》。

描写男欢女爱、离愁别绪的传统题材：

  • 南渡前追求的是轻狂放逸的人生境界；
  • 靖康之难后抒发国破家亡的沉痛之情、晚年隐居写了大量隐逸词。

词风为独创的 “樵歌体”：

  • 淡而静的空旷境界中，透出潇洒；
  • 风格自然飘逸、语言浅白如话，自成一格。

《西江月》

世事短如春梦，人情薄似秋云。不须计较苦劳心。万事原来有命。

幸遇三杯酒好，况逢一朵花新。片时欢笑且相亲。明日阴晴未定。

# 辛派词人

# 辛弃疾

他代表了南宋爱国词的最高成就，达到了词体艺术的高峰，发出了时代最强音，形成 “稼轩体”。著有《稼轩词》，又名《稼轩长短句》。

题材内容：

  • 抒发报国豪情和壮志难酬的悲愤。如《鹧鸪天》《破阵子》
  • 壮志难酬、报国无门的愤懑中注入急切。如《水龙吟・登建康赏心亭》《摸鱼儿》
  • 缠绵之情。《青玉案・元夕》
  • 农村田园的生活和景致。如《清平乐》《西江月・夜行黄沙道中》

艺术成就：

  • 豪放词刚柔并济，婉约词柔中有刚，悲壮中有婉转，豪放中有缠绵，柔情中有刚劲道，是辛弃疾词风的独特之处；
  • 喜好议论、以论为词。用典广播，且典源广泛；
  • 以文为词，采用散文句法，创造用散文句入词。

《菩萨蛮》

郁孤台下清江水，中间多少行人泪？西北望长安，可怜无数山。
青山遮不住，毕竟东流去。江晚正愁余，山深闻鹧鸪。

# 张元干

词作特点：

  • 前期：“极妩秀之致”；
  • 后期：慷慨激昂
# 张孝祥

其词反映时代特征，极具艺术感染力。南宋周密《绝妙好词》收集的南宋词中，列为第一。

# 陈亮

特点：以救国安民为己任，表现抗战恢复的之气

# 刘过

特点：多壮语，有稼轩词狂放而又幽默的影子，风格、结构都与辛弃疾相似

# 南宋中期诗文

南宋中期是宋代文学发展的又一高峰，出现 “中兴四大诗人”。

# 陆游

古代作家中最高产的诗人，有《剑南诗稿》《渭南文集》。

创作时期：

  • 早期：喜藻绘而至清新脱俗；
  • 中期：豪放悲壮；
  • 晚期：清淡秀逸，风格闲适淡泊，但爱国思想和积极奋斗的精神继续保存。

艺术成就：

  • 古体诗风格悲壮；
  • 律师多悲愤之作；
  • 绝句风格清淡秀逸。

陆游词和散文：

  • 词作风格多样，其中激昂慷慨、清淡秀逸、流利绵密为主；
  • 《渭南文集》以记叙文成就最高：《老学庵笔记》《入蜀记》
# 杨万里

著有《城斋集》。

城斋体：

以日常生活的小情趣为题，想象新奇风趣、语言通俗明快、风格流转圆活的特点。

活法诗：

以自然意象为主，恢复诗人感官的天真状态，表现山水景观所蕴含的自然灵性和知觉情意，诗中充满奇趣和活劲。

《小池》
泉眼无声惜细流，树阴照水爱晴柔。
小荷才露尖尖角，早有蜻蜓立上头。

# 范成大

著《石湖居士诗集》。

诗歌：

  • 记事诗
    • 抒写瞻望收复河山的心怀和壮语；
    • 鞭挞统治者的卖国屈辱。

田园诗艺术特点：

田园诗和 “田园词”、“悯农诗” 合二为一。

《夏日》（其七）

昼出耘田夜绩麻， 村庄儿女各当家。

童孙未解供耕织， 也傍桑阴学种瓜。

# 朱熹

著《四书章句集注》《诗集传》《楚辞集注》

诗歌：以诗喻理，情趣、理趣相交织。语言净洁简理，句法自然平易，风格雅正明洁。

散文：功力深刻、理致周密，不矜才使气看来诗平淡无奇，却深厚简单，含不尽之意于言外。

《春日》

胜日寻芳泗水滨，无边光景一时新。
等闲识得东风面，万紫千红总是春。

# 其他散文

中期文坛分为两派：

  • 一派是讲事功的功力派，以陈亮和叶适最为有名，其文章以评论时事和讲乱兴衰为特色；
  • 另一派是道家派，以朱熹、真德秀为代表，多讲学之文和语录体。

个体散文的一般特征：

  • 议论文：以扶贫救倾的议论文成就最高，正气贯注，文字最为明达，。受欧、曾影响，南宋散文文字风格更趋向畅达明快和语体化，但也有追求平易而流为冗慢与粗率的弊端，导致文气冗若。
  • 叙事文：叙事性文章显得纡漫拖沓，碑志比较散漫萎弱。
  • 语录体：理学家以无意为文，无章法约束，过于随意，不以文章论

# 南宋清雅词派

# 姜夔

首创了清雅词派，丰富了词体风格内涵，形成 “第三派”。使词能够与传统诗文并列并肩，从传统《花间》、南唐、晏殊、柳永、周邦彦所形成的婉丽传统风格中独立出来。

诗学批评方面《诗说》，见解读到而深刻。

词的创作成就最高，影响最大。有《白石道人诗集》，词集名《白石道人歌曲》

情思内涵：

  • 题材上：以感时伤世，咏物言志和追记恋爱恋情为主。
  • 音调和意境上：有情越、高旷的格调，词中塑造的形象往往融入了他本人所具有的高雅气质。
  • 还有反映家国之感、时政感慨的作品，不以描写的真切、感情的激愤取胜，而是让自己的感情上笼罩上凄清的色彩。

艺术成就：

继承周邦彦，与周邦彦合成 “周姜”。清空、骚雅成为白石词独特风格的代称。

  • 清空：清空与质实相对。
    • 清幽空灵的意境。多用素洁的意向，营造清疏高旷的境界。如《暗香》
    • 虚字的使用。使语意转折灵活，流走自如，而又传神入微，且能避免平铺直叙的缺点。如《疏影》
    • 独特的笔法。以柔笔写柔情，并能褪尽铅华，更见清刚。如《长亭怨慢》
  • 音律精严，词中求雅。以雅乐注入词体，主要有两种方法：
    • 以古乐府入词，用古乐的雅音革除今曲的淫靡；
    • 以唐法曲音乐注入词中，使 “清”、“空”、“淡” 的风格代替胡乐的浓艳急促。

《扬州慢》

淮左名都，竹西佳处，解鞍少驻初程。过春风十里，尽荠麦青青。自胡马窥江去后，废池乔木，犹厌言兵。渐黄昏，清角吹寒，都在空城。
杜郎俊赏，算而今、重到须惊。纵豆蔻词工，青楼梦好，难赋深情。二十四桥仍在，波心荡、冷月无声。念桥边红药，年年知为谁生。

# 吴文英

主要效法周邦彦，就追求词的典雅含蓄而言，收姜夔影响较大，因此常把他列为周派词人，著《梦窗词》。

题材：

  • 酬酢赠答之作；
  • 哀时伤世之作；
  • 忆旧悼亡之作；

艺术成就：

  • 讲究用典和用字，形成雅丽风格并有独特的情绪体验和表现方式；
  • 章法结构十分独特，时间和空间交错杂糅的叙述方式与亦真亦幻的内容呼应；
  • 语言有自己风格，运用冷僻怪异的字词，形成超逸沉博、浓艳密丽的词风；

《风入松》

听风听雨过清明，愁草瘗花铭。楼前绿暗分携路，一丝柳，一寸柔情。料峭春寒中酒，交加晓梦啼莺。

西园日日扫林亭，依旧赏新晴。黄蜂频扑秋千索，有当时、纤手香凝。惆怅双鸳不到，幽阶一夜苔生。

# 史达祖

“梅溪先生”，咏物词善于取神、用意，著《梅溪词》。

其词词语奇秀清逸，融情景与一家，会句意于两得。

# 周密

著《草窗词》。

词风与姜夔相近，和吴文英起名，并称 “二窗”。

其词风格 “清丽”，咏物词寄托遥深。

# 王沂孙

著《碧山乐府》（又名《花外集》）。

其词别具悼念故国之思，充满凄苦悲凉意味。

其咏物词，既体物细微，结构细密，又寄托遥深，哀婉动人，惜时有隐晦之病。

# 张炎

“张春水”、“张孤雁”，著《词源》二卷。

上承白石的清空疏宕，近与梦窗的密丽质实区别，表现了明确的风格意识。

创作风格：

  • 宋亡之前：词风雅丽；；
  • 宋亡之后，词风凄凉悲苦；
  • 入元后，词风转变，多为哀音及身世零落之感；

《千秋岁》

数声鶗鴂。又报芳菲歇。惜春更把残红折。雨轻风色暴，梅子青时节。永丰柳，无人尽日飞花雪。
莫把幺弦拨。怨极弦能说。天不老，情难绝。心似双丝网，中有千千结。夜过也，东窗未白凝残月。

# 南宋后期文学

南宋后期因为政治动荡黑暗，导致文学的发展每况愈下。“永嘉四灵”、“江湖诗派是主要代表。宋亡之际文天祥等爱国诗人是这一时期的最后一道光芒。

# 永嘉四灵

徐玑，号灵渊；

徐照，子灵晖；

翁卷，字灵舒；

赵师秀：字灵秀；

特征：

  • “四灵” 作诗复兴唐诗实际是晚唐体，专做格律诗；
  • 意平语诡，多有怆气，因刻画太甚流于纤仄，也有自然浑圆的作品；
  • 攻五言格律，多咏景物，写萧散野逸之趣，追求平淡简远的韵调；
  • 费心苦吟，刻意求工，其诗冲淡平和、清零清寒的境界，流露山人名士的出世情怀；
# 刘克庄

著《后村先生大全集》，词集名《后村长短句》《后村别调》

继承辛派爱国注意传统与豪放词风，是辛派词人的后劲。

作词针对社会时政有感而发，内容多是忧愤悲凉，黍离哀痛的沧桑感。

创作阶段：

  • 前期：从晚唐体、四灵体乳瘦，刻琢精丽，风格清轻简淡。后追求抑扬开阖、悲愤慷慨的作风；
  • 后期：有意追求陆游、杨万里的大家风度，格调更多沉痛、感情更多深刻；
# 戴复古

作诗以苦吟秋工，带有四灵余习。著《石屏集》。

# 方岳

有《秋崖集》。

作诗主清新，工于镂琢，常能制作出新奇对偶，把寻常典故成语巧加组织而成名句。

大多数作品写于罢官乡居时，颇多写景的田园之作。

# 文天祥

著《文山先生全集》。

创作阶段：

  • 早期：诗歌比较平庸；
  • 后期：
    • 诗歌：抒发爱国思想情感与坚贞的民族气节，多谢亲身经历，具有纪实性，直抒胸臆，悲壮苍凉；
    • 散文：以纪实著称，富有惊心动魄的传奇性；
    • 词：创作不多，表现了作者至死不渝的民族气节和顽强斗志。
# 汪元量

著《水云集》《湖山类稿》。

创作：

  • 诗：以纪实绝句述亡国之痛，充满了亡国之戚、去国之苦。被称为宋亡 “诗史”；
  • 词：直抒胸臆，多写故国之思，常用比兴而不流于隐晦，着笔疏淡而不失于苦寂；
# 刘辰翁

著《须溪集》。

其词大都为宋亡后作，感怀时事，抒发兴亡之感。

诗文以奇怪磊落为宗，不平的心境和抑郁的情绪，使其诗遣词造句显得很艰，意趣迷离，寄托遥深。

# 郑思肖

著《所南翁一百二十图诗集》《郑所南先生文集》。

其诗多写亡国之痛与故国情怀。
