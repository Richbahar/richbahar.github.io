# 一、运营是什么？

在《运营之光》中，有一句话的印象特别深刻：

过去十年是产品驱动的时候，未来十年是运营驱动的时候。

那么运营到底是什么？

可能对于刚从事这一行的萌新而言，运营是一个啥事都做的岗位，对运营目的含混不清。

静秋老师说：运营就是为了更好的完成公司目标而存在的岗位。

我认为这个定义下得十分精确简练。对于运营而已，日常工作无非就是围绕着品牌或产品（服务）而展开的，而这一切工作的目的最终将归根到公司的收益目标上。

运营这个岗位几乎覆盖了现代企业的整个发展周期，从产品原型的设计到产品的研发及最终的推广都是需要运营的参与，差别不过是参与程度的轻重而已，下面将展开来阐述。

# 产品原型期：运营中度参与

这个阶段主要是产品经理的工作。
运营所参与的部分是协助产品经理做产品调研（用户和市场）。

# 产品开发期：运营轻参与

这个阶段注意是开发人员的工作。
运营在这个阶段是配合开发做灰度发布和 A/B 测试。

# 产品运营期间：运营重参与

这个阶段是运营人员的主要工作。
在灰度测试和 A/B 测试后，运营就会围绕着这个产品开始做运营优化，然后需求 BD 合作，最后把产品推广给所有的人。

# 二、运营有哪些分支？

就传统意义上的运营而言，大致分为了内容运营、用户运营、活动运营、数据运营几类。

# 内容运营

内容运营的工作主要是负责生产目标用户所接纳的包括文字、视频、图片等形式的内容，围绕着内容的生产和消费搭建一个良性循环。

内容运营日常工作可能是这样的：

  • 我的内容定位是什么？
  • 谁会点开我的内容？
  • 我的文章风格是怎么样的？
  • 我的内容从哪里来？
  • 内容是否能让用户愿意消费 / 传播？
  • 如何引导用户看完 / 评论我的内容？
  • ……

内容运营是运营很重要的组成部分，包括现在的新媒体运营、抖音运营、文案运营等应当都是属于这个范围内。

简单的来说，内容运营就是创造用户愿意消费 / 传播的内容。

# 用户运营

用户运营就是以用户为中心的运营方法。围绕用户的获取 — 留存 — 转化及用户间的互动建立良性循环。

用户运营日常工作可能是这样的：

  • 我的用户是谁？
  • 用户从什么渠道来的？
  • 如何建立与用户的联系？
  • 如何让用户愿意消费 / 留存？
  • 如何提高对大量用户的管控？
  • 用户流失机制的如何建立？
  • ……

用户运营就和社群运营、论坛运营、粉丝运营是一个性质的。

简单来说：用户运营就是让产品与用户、用户与用户建立联系。

# 活动运营及数据运营

这两个运营岗位属于所有运营必须掌握的基础能力，一般在大公司或有专门需求的公司单独设定。

主要是负责活动策划和执行、数据的采集和分析。

# 三、运营产品的差异

以下属于《运营之光》的笔记

# 工具栏产品

  • 注重效率和体验，产品大雨运营
  • 运营的关注点事用户增长
  • 主要推广手段：渠道推广、BD 和部分活动
  • 运营跟数据打交道比较多

# 社交 / 社区类产品

  • 注重社交氛围、话题和玩法，需要运营和产品并重

  • 运营是经常与人打交道：

    • 选择符合产品调性的用户使用产品
    • 后续持续做好重点用户维护
    • 自己深入产品中带着用户一起玩

  • 在产品氛围的维护上，运营还需要制造各种话题和策划各种活动

    • 确保话题和活动都是目标用户喜欢的，所以运营最好深度理解用户
    • 自己本身就是产品的目标用户

  • 用户增长除了常规的渠道、引流和跟数据打交道外，可能也需要经常通过制造一些事件、话题、传播等来完成拉动。

# 内容类产品

  • 持续做出独特、高质量的好内容
  • 包装内容，让用户更易于消费（取一个好的标题、做一个好的专题等）
  • 用户增长依托于内容和话题
  • 需要定期策划出很多有亮点的话题性内容，借助内容传播拉来更多用户

# 电商类产品

  • 产品和品类的运营
    • 选择一些什么样商品品类来售卖
    • 售卖过程中去哪里推哪几款商品
    • 商品的定价策略如何制定
    • 商品的库存管理和供应链管理
  • 促销活动的策划和落地执行，比如：6.18、11.11、12.12
  • 推广和流量建设，与数据打交道，需要熟悉各种推广渠道和推广工具
  • 用户关怀和用户维系的东西，针对什么样的用户定期送小礼物、代金券等
  • 基础东西：商品的包装、文案优化、客服到位等

# 平台类产品

  • 特别注重 “节奏”，也就是 B 端和 C 端的平衡性问题
  • 注重策略和用户维系，在成熟阶段，针对用户的不同特征精细化运营。如：淘宝商家。可能按地区、按售卖商品、接客单价等不同纬度分成很多类，然后分别对其进行维系
  • 渡过发展临界点后，非常注重品牌，各类活动、品牌宣传文章、事件、内容等策划落地成为重点

# 游戏类产品

目前游戏产品的运营，重点是两大块：

  • 推广：对接渠道、看转化率、盯数据
  • 收入：围绕土豪运营

# 四、相关术语

A/B 测试

为产品页面或流程制作 2 个或多个版本，同一时间让具有同类特征的用户随机访问不同的版本。然后收集意见、评估出最好的版本，正式录用。

BD（商务拓展）

运营中的 BD，是根据自身产品或服务特性选择合适的对象合作，促进提升双方商业利益。

CPC（每次点击费用）

投放的广告单次被点击的收费。投放者只需要对广告被点击收费，如果只是浏览而没有点击则不收费。

CPM（每千次展现收费）

广告案显示次数（或弹出次数）计算广告费，以 1000 次显示为一个计价单位。

PV（页面浏览量）

用户浏览数。同一个用户，点击一次就算一次 PV。就像我主页中那个小眼睛后的统计就属于 PV。
